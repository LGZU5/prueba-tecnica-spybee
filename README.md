# PRUEBA TECNICA SPYBEE

Aplicación hecha por Jesus Tovar para postulación a puesto de desarrollador FrontEnd

## Características

- **Gestión de proyectos** con vista de lista y mapa interactivo
- **Visualización geográfica** con Mapbox GL JS
- **Filtrado y búsqueda** en tiempo real
- **Panel lateral dinámico** con estadísticas y resumen
- **Seguimiento de incidencias**, RFIs y tareas
- **Paginación inteligente** con navegación optimizada
- **Diseño responsive** adaptado a desktop, tablet y mobile
- **Interfaz moderna** con animaciones fluidas

## Tecnologías

- **Framework:** Next.js 14 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** CSS Modules
- **Estado:** Zustand
- **Mapas:** Mapbox GL JS
- **Despliegue:** Vercel

## Prerequisitos

Tener instalado:

- **Node.js** versión 18.x o superior ([Descargar aquí](https://nodejs.org/))
- **npm** o **pnpm**
- **Git**
- Una cuenta en **Mapbox** para obtener el token de API ([Crear cuenta](https://www.mapbox.com/))
